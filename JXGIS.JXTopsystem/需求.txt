1、如果选择了不制作门牌，就可以不用再勾选需不需要邮寄门牌了
2、点了邮寄门牌，邮寄地址必须要填
3、默认制作门牌和邮寄门牌都是选中的
4、在新增门牌表单没有填完数据进行过保存，就不允许点击门牌证打印和地名证明开具的按钮，保存成功后才可以使用这两个按钮
5、住宅门牌必填项：市辖区、镇街道、小区名。住宅门牌如果填了门牌号这个字段后，门牌规格必填
6、道路门牌必填项：市辖区、镇街道、道路名、门牌号、门牌规格
7、农村门牌必填项：市辖区、镇街道、村社区、自然村名称、门牌号、门牌规格

8、查询条件的都可以分成这几类：
             行政区划[调接口 Common/GetUserDistrictsTree()]、
			 小区名（道路名、自然村名）[调接口Common/getNamesByDistrict(string CountyID, string NeighbourhoodID, string CommunityID, int MPType)]、
			 门牌号码（针对道路门牌单双号）、
			 地址编码、
			 产权人、
			 标准地址

9、门牌修改时需要把标准地址拼接好给我，新增时可以不给，我在后台拼接
   样子：var StandardAddress = CountyName 
							  + NeighborhoodsName 
							  + CommunityName 
							  + newData.ResidenceName 
							  + newData.MPNumber == null ? "" : newData.MPNumber + "号" 
							  + newData.Dormitory 
							  + newData.LZNumber == null ? "" : newData.LZNumber + "幢" 
							  + newData.DYNumber == null ? "" : newData.DYNumber + "单元" 
							  + newData.HSNumber == null ? "" : newData.HSNumber + "室";）

10、字典类：
			门牌规格接口：Common/GetMPSizeByMPType(int mpType)   1住宅  2 道路   3农村
			根据道路名获取道路的起点、讫点和编制规则接口：Common/GetRoadDic(string CountyID, string NeighborhoodsID, string CommunityID, string RoadName)
			邮政编码接口：Common/GetPostcodeByDID(string CountyID, string NeighborhoodsID, string CommunityID)
